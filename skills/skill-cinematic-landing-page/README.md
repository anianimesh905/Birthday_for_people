# Cinematic Landing Page Generator Skill

## Overview
A reusable skill for generating deeply atmospheric, art-directed landing pages with:
- Multi-plane parallax heroes
- Editorial typography & scrapbook photo clusters
- Glassmorphism UI panels
- Scroll-triggered stagger animations
- Custom cursor & smooth scroll

## File Structure
```
skill-cinematic-landing-page/
└── SKILL.md          # Complete skill documentation
```

## How to Use
1. Copy `SKILL.md` into your project's `.agents/skills/` directory
2. Reference it when asking for cinematic landing pages
3. The skill provides: design tokens, component patterns, motion specs, and implementation notes

## Key Design Principles
- **Zero stock-travel-blog energy** — every surface is considered
- **Cinematic pacing** — fade to black between sections
- **Asymmetric editorial layouts** — scrapbook photo clusters, hairline rules
- **Generous whitespace** — the page breathes
